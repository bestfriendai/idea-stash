# IdeaStash Testing Guide

## Manual Testing Checklist

### Core Functionality
- [ ] App launches without crash
- [ ] Sample ideas load on first launch
- [ ] Ideas display with all information
- [ ] Category filtering works
- [ ] Search filters correctly
- [ ] Pull-to-refresh works

### Favorites
- [ ] Tap star toggles favorite
- [ ] Favorites persist after restart
- [ ] Favorites tab shows correct ideas

### Categories
- [ ] All 8 categories available
- [ ] Category colors display correctly
- [ ] Category chips filter correctly

### Tags
- [ ] Tags display on idea cards
- [ ] Search finds tags

### CRUD Operations
- [ ] Add new idea works
- [ ] Edit idea works
- [ ] Delete idea works

## Device Testing
- [ ] iOS Simulator
- [ ] Android Emulator

## Performance
- App launch: < 2 seconds
- Screen transitions: < 300ms
